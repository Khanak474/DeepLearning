# 📈 Stock Price Analysis & Prediction using LSTM

## 📌 Overview
This project focuses on **stock trend visualization and prediction using LSTM (Long Short-Term Memory)** models. It combines data analysis, visualization, and deep learning to understand and forecast stock price movements.

The notebook is structured into modular sections covering:
- Data processing
- Visualization
- Model building
- Company-specific analysis

---

## 📂 Project Structure

### 🔹 Importing Libraries
- Essential Python libraries for:
  - Data handling (NumPy, Pandas)
  - Visualization (Matplotlib)
  - Deep Learning (TensorFlow/Keras)
  - Stock data fetching (yfinance)

---

### 📊 Graph Plotting Functions
- Custom reusable functions for:
  - Plotting stock price trends  
  - Comparing multiple companies  
  - Visualizing predictions vs actual values  
- Helps maintain clean and modular code  

---

### 🧠 Building and Training LSTM Model
- LSTM model created for time-series forecasting  
- Steps involved:
  - Data normalization  
  - Sequence creation (time window input)  
  - Model architecture design  
  - Training using historical stock data  

---

### 📉 Visualizing Stock Trends Across Companies
- Comparative analysis of multiple companies  
- Trend visualization to understand:
  - Market behavior  
  - Price movement patterns  
- Useful for identifying correlations and differences  

---

### 🔍 LSTM-Based Analysis of INOXWIND
- Focused case study on **INOXWIND stock**  
- Includes:
  - Data preprocessing  
  - Model training  
  - Prediction visualization  
- Demonstrates real-world application of LSTM
